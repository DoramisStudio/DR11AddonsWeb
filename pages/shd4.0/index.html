OVERVIEW

SHD Series V4.0

Update ke 4.0 untuk Addons Jetbus 3 All In One lite untuk Trainz Simulator!

Satu set Jetbus SHD series, mulai dari Jetbus 2, jetbus 3, dan Jetbus 5 yang bisa dikustomisasi sendiri seperti selendang, topi, muka, acc, dll. belum termasuk bagian belakang (V4.0).

FEATURES

  • Mudah direskin (Manual replacement)
  • Model Face (Face JB2 - JB5)
  • Model Topi (Topi JB2 - JB3)
  • Pembaruan Face JB3, model lampu lebih bagus dan dapat dilepas.
  • Medium Poly, ringan dan ramah performa.
  • Accesories bervariasi.

MINUS

  • Belum ada script
  • Roda belum ber-anim
  • Strobo belum ber-anim (sepaket dengan pembuatan script)

COMPATIBILITY

Compatible with:
  • Trainz Simulator 2009 (Minor issue)
  • Trainz Driver 2, atau TSA (Minor issue)
  • Trainz Simulator 2012
  • Trainz: A New Era
  • Trainz Railroad Simulator 2019
  • Trainz Railroad Simulator 2022
